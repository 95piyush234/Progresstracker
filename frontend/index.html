How To Use Progress Tracker Pro

A visual walkthrough for new users

Understand what the website is for, how to log progress, and where to see your graphs, history, proof uploads, and backups.

What this website is for

Track absolutely anything in one backend-connected workspace.

Use this website to manage study plans, game ranks, workouts, reading goals, habits, projects, or any custom milestone. You create trackers, log updates manually, attach proof images, and review everything from dashboard, graphs, and history.

Custom trackers Manual quick logs Proof image uploads Visual graph view Local-first backup
1. Dashboard

See your whole workspace at a glance

Start on Dashboard to check total trackers, today’s progress, due milestones, overall completion, recent movement, and the strongest category in your workspace.

2. Create Tracker

Build any custom tracker you want

Use Create Tracker to define a title, category, unit, target, deadline, color, icon, notes, creator name, and custom fields that match your goal.

3. Quick Log + Proof

Open a tracker, add progress, then attach proof

The detail drawer lets you update amount, date, tag, and note. You can also upload a screenshot or photo as proof, then save it so it appears in Timeline and History.

4. Graph View

Use the Graph page for visual progress insights

Open Graph to compare trackers, review weekly movement, read category performance, and understand your progress with clearer visual boards.

5. History + Backup

Review old entries and keep local backups

Use History to find every log, correction, note, and proof image. Export JSON whenever you want a backup or import a saved snapshot to restore your workspace later.

Best flow

Recommended first-use path

  1. Open the sample trackers and see how they are structured.
  2. Create your own tracker with custom labels and units.
  3. Use Quick Log to save progress regularly.
  4. Upload proof images when you want visual evidence.
  5. Visit Graph and History to review momentum over time.
